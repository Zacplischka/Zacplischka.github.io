Project Overview

This project analyzes AFL player statistics from 2015-2025, providing insights into player performance, team dynamics, and league trends.

Features:

  • Interactive dashboard with real-time filtering
  • Player performance comparison tools
  • Team statistics and analysis
  • Historical trend visualization
  • SuperCoach pricing integration
  • Comprehensive data cleaning and processing pipeline
